# Nodejs Order API.

> A Node.js project

## Build Setup

``` bash

# Clone this repo
$ git clone https://github.com/soyserkan/order-api.git
$ cd order-api

#Create .env file
You can find the example .env file in the root directory (.env-example)


# install dependencies
npm install

# serve with hot reload at localhost:3000
npm run start:dev
```
## Prerequisites
- MongoDB
